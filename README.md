
# Health Prediction Application

## Overview

This project is an AI-powered Health Prediction Application developed using Python, Streamlit, SQLite, and Machine Learning.

The application allows users to manage patient records, predict potential health risks using blood test parameters, and store prediction results.

---

## Features

- Create Patient Records
- View Patient Records
- Update Patient Records
- Delete Patient Records
- Data Validation
- SQLite Database Storage
- Machine Learning Prediction
- User-Friendly Streamlit Interface

---

## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- Scikit-Learn
- Joblib

---

## Dataset

The dataset contains:

- Full Name
- Date of Birth
- Email
- Glucose
- Haemoglobin
- Cholesterol
- Disease Label

---

## Machine Learning Model

Random Forest Classifier was used for health risk prediction.

Possible outputs:

- Healthy
- Diabetes Risk
- Heart Disease Risk

---

## Installation

pip install -r requirements.txt

python train_model.py

streamlit run app.py
